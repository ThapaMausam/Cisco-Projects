Network Address = 192.168.1.0
Subnet Mask = 255.255.255.0 or 11111111.11111111.11111111.00000000

No. of subnets (N) = 3
Then,
    2^n >= 3 where, n = no. of burrowed bits from Network
or, n = 2

So,
New Subnet mask = 11111111.11111111.11111111.11000000
OR in Decimal, Subnet Mask = 255.255.255.192

# Block Size for Subnet Mask ending with .192 is 64
